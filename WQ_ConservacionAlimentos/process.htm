O Proceso    

Para levar a cabo a tarefa, debes seguir os seguintes pasos:

  1. En primeiro lugar, lembra que eres un/unha investigador/a quen se lle pide unha información IMPRESCINDIBLE para sobrevivir todo un inverno.
  2. Lembra todo canto sabes de como se conservan os alimentos na túa casa ou na túa familia. Seráche moi útil.
  3. A continuación, le con atención os documentos web que se che facilitan
  4. Así conseguirás saber cal é o proceso máis axeitado de conservación para cada un dos alimentos.Sen repetir. Isa é a primeira TAREFA.
  5. Propón tamén un procedemento alternativo para cada un.
  6. Lembra que non podes contar coa corrente eléctrica. Pero sí teñen sal, especias, lume, frío natural, vento...
  7. Escribiráslles unha carta coas túas conclusións.
  8. Debes facelo antes de final do mes.
